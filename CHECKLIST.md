# ✅ Setup Checklist

Use this to track your setup progress:

## Pre-Setup
- [ ] Node.js 18+ installed
- [ ] Firebase account created (free)

## Firebase Console Setup
- [ ] Created new Firebase project
- [ ] Copied Firebase config object
- [ ] Enabled Email/Password authentication
- [ ] Created Firestore database
- [ ] Set Firestore security rules

## Local Setup
- [ ] Downloaded/extracted project files
- [ ] Ran `npm install`
- [ ] Pasted Firebase config into `src/App.jsx`
- [ ] Ran `npm run dev`
- [ ] App loads at http://localhost:3000

## First Use
- [ ] Created first user account
- [ ] Went to Firestore → users collection
- [ ] Set `isAdmin: true` for your user
- [ ] Refreshed app and saw ADMIN tab
- [ ] Updated game matchups in admin panel
- [ ] Made test picks
- [ ] Verified leaderboard works

## Going Live (Optional)
- [ ] Installed Firebase CLI (`npm install -g firebase-tools`)
- [ ] Ran `firebase login`
- [ ] Ran `npm run build`
- [ ] Ran `firebase deploy --only hosting`
- [ ] Tested live URL
- [ ] Shared URL with friends

## Tips
- Update game matchups as soon as playoff bracket is set
- Enter game results immediately after each game
- Check mobile view - it's optimized for phones!
- Set a deadline for picks before Wild Card weekend
